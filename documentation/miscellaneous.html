setAssignMethod
setAssignMethod(obj: undefined, propertyName: undefined, writeOnce: undefined)

Create a function for setting a value for a property on a given object.

Parameters :
  • obj

    The object to apply the key & setter on.

  • propertyName

    The name of the property on the object

  • writeOnce

    If true will allow writing once (default: false)

    Example: let obj = new FluentAssign; setAssignMethod(obj, 'myProp'); obj.myProp('someValue'); const result = obj.toJSON(); console.log(result); //{ myProp: 'someValue' }

    let obj = new FluentAssign; setAssignMethod(obj, 'myProp', true); // applying writeOnce obj.myProp('someValue'); obj.myProp('someValue'); // ERROR: Overriding config property 'myProp' is not allowed.
setAssignAlias
setAssignAlias(obj: undefined, propertyName: undefined, srcPropertyName: undefined, hard: undefined)

Create a function for setting a value that is an alias to an other setter function.

Parameters :
  • obj

    The object to apply the key & setter on.

  • propertyName

    The name of the property on the object

  • srcPropertyName

    The name of the property on the object this alias points to

  • hard

    If true, will set a readonly property on the object that returns the value of the source property. Default: false

    Example: let obj = new FluentAssign ; setAssignMethod(obj, 'myProp'); setAssignAlias(obj, 'myPropAlias', 'myProp'); obj.myPropAlias('someValue'); const result = obj.toJSON(); console.log(result); //{ myProp: 'someValue' } result.myPropAlias // undefined

    let obj = new FluentAssign ; setAssignMethod(obj, 'myProp'); setAssignAlias(obj, 'myPropAlias', 'myProp', true); // setting a hard alias. obj.myPropAlias('someValue'); const result = obj.toJSON(); console.log(result); //{ myProp: 'someValue' } result.myPropAlias // someValue

src/ngx-modialog/src/models/overlay-context.ts

overlayConfigFactory
overlayConfigFactory(context: undefined, baseContextType: undefined, baseConfig: undefined)

A helper to create an OverlayConfig on the fly. Since OverlayConfig requires context it means a builder is needed, this process had some boilerplate. When a quick, on the fly overlay config is needed use this helper to avoid that boilerplate.

A builder is used as an API to allow setting the context and providing some operations around the modal. When a developers knows the context before hand we can skip this step, this is what this factory is for.

Parameters :
  • context

    The context for the modal

  • baseContextType

    Optional. The type/class of the context. This is the class used to init a new instance of the context

  • baseConfig

    A base configuration that the result will extend
